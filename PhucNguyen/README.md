# Phuc Nguyen

# Instructions

## 1. Clone repo from Github

Run the following command:

```bash
gh repo clone phucprime/int-quiz
```

## 2. Install dependencies

```bash
cd PhucNguyen
```

```javascript
npm install
```

## 3. Build on a device

Android OS:

```bash
npx react-native run-android
```

IOS:

```bash
npx react-native run-ios
```
