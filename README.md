# Unitz Test

Bản này được dev trên mac nên hiện tại đang support cho mac.

## Setup Guide.

- Clone cái repo về.
- Chạy `yarn install`. Bản này auto link rùi không cần chạy `react-native link`.

### For iOS

- Vào thư mục `ios` chạy `pod install`.
- Chạy `yarn ios` để chạy app.
